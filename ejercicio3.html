Ejercicio 3

Este programa permite el ingreso de numeros hasta que el usuario desee. Para ingresar un número debe ingresarlo en el campo Número y luego presionar el botón Ingresar Datos, al presionar este botón los numeros serán almacenados en un vector, el cual se irá visualizando en el campo de Vector ingresado. Para ver el vector ordenado de forma ascendente debe seleccionar la opción ASC y luego presionar el botón Ordenar Vector, para ver el vector ordenando de forma descendente debe seleccionar la opción DSC y luego presionar el botón Ordenar Vector.
